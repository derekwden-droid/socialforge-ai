# SocialForge AI

**Unified Audience Engagement Graph Platform — 13 AI Subsystems for Autonomous Social Growth**

SocialForge AI turns your social data from **X, YouTube, Instagram, Facebook, and TikTok** into a live interactive command center. Predict virality, surface your most interactive followers, and get Claude-powered growth plays that drive real comments, DMs, saves, and shares.

**Built for**: Influencers, creators, agencies, and brands who want engagement — not just vanity metrics.

**Supported Platforms** (MVP uses realistic simulated data — real OAuth in v1.1):
- X (Twitter)
- YouTube
- Instagram
- Facebook
- TikTok

## Quick Start
```bash
npm install
cp .env.example .env          # optional — only for AI Engine
npm run dev
```
App runs at http://localhost:5173

## Features
- Live Unified Audience Engagement Graph (UCIG) with drag interaction
- 13 AI subsystems tailored for social growth
- Claude AI Engine for 24-hour growth plays
- Auto-refreshing charts + fully responsive mobile drawer
- Error-isolated views
